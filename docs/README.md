# Back End Node.js

## O Curso

- Back End Node.js
  - Router, Middleware, Autenticação, Validação, Stream, Docker e mais.

- Pré-requisitos
  - JavaScript, TypeScript e SQLite

- Ferramentas
  - VS Code e Node 24+

---

## Projeto

Esse é um curso de Projeto, onde um Framework e um LMS (Learning Management System) serão desenvolvidos do início ao fim.

### Zero Dependências

Um Framework de Back End em Node.js será criado do zero para o desenvolvimento de APIs. O Framework não terá nenhuma dependência.

### Projeto a ser construído: [Tiny LMS](https://lms.lobo.api.br/)

---

# Grade

1.  Back End Node.js
    - Back End e Ferramentas

2.  Servidor Simples
    - createServer, Request, Response, Router, FS, Banco de Dados

3.  Framework
    - Router Dinâmico, Middleware, Database, Organização

4.  API
    - Tabelas, Rotas, Cursos, Aulas, Certificados

5.  Autenticação
    - Session, Hash, SHA256, Scrypt, Auth Middleware, Reset Token, Password

6.  Validação
    - Body Parser, Tipar, Formatar, Validar, Rate Limit

7.  Stream
    - Memória, Readable Stream, Writable Stream, Upload de Arquivos

8.  Front End
    - HTML, CSS, Login, Admin, Cursos, Certificados, Vídeo, Paginação

9.  Produção
    - Docker, Docker Compose, Volumes, Caddy, CORS, Headers, Subprocessos, Email, SSH, Hospedar

10. Considerações Finais
    - Bibliotecas, Frameworks e Conclusão
